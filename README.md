# Business Sales Performance Analytics

## Project Overview

This project analyzes business sales performance using Python and data visualization techniques.

The objective is to identify:

* Sales trends
* Top-selling products
* Profitable regions
* High-performing categories

---

## Dataset Used

Sample Superstore Dataset

---

## Tools & Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / VS Code

---

## Charts Created

* Category Distribution
* Sales Trend Over Time
* Top 10 Products
* Region-wise Profit Analysis

---

## Key Business Insights

1. Technology category generated the highest revenue.
2. West region achieved the strongest sales performance.
3. Some products contributed major revenue.
4. Revenue trends showed growth in several months.
5. High-performing categories can be targeted for future growth.

---

## Conclusion

This project demonstrates how sales data can be analyzed to generate meaningful business insights and support better decision-making.
